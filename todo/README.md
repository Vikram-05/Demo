# Modern Todo App

A responsive todo web application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks with due date and priority
- Mark tasks complete or active
- Edit task text inline
- Delete individual tasks
- Clear completed tasks
- Delete all tasks
- Search, filter, and sort tasks
- Drag-and-drop reorder
- Persist data in browser local storage
- Light/dark theme toggle

## How to use

1. Open `index.html` in a browser.
2. Add tasks in the form and press `Add Task`.
3. Use filters, search, or sort to manage your list.
4. Completed tasks remain saved between sessions.
